# Projects Portfolio

A curated collection of my completed and ongoing projects across **Product Analytics**, **Machine Learning Research**, and **Data Engineering**.

This repository is meant to showcase not only the final outputs, but also the kinds of problems I like working on: metric decomposition, experimentation, reproducible research, representation learning, search systems, and end-to-end ML pipelines.

* **Product Analytics:** metric decomposition, root-cause analysis, A/B interpretation, market exploration, and business-oriented evaluation.
* **ML Research:** NLP, representation learning, probing, knowledge graph completion, audio modeling, and model tuning.
* **Data Engineering:** search pipelines, distributed processing, workflow orchestration, and containerized ML deployment.
